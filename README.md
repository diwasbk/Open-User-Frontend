# Open User Frontend

A React-based frontend application for managing user profiles. This project allows you to **add, view, update, and delete users** through a clean and responsive UI. It works with the [Open User API](https://github.com/diwasbk/Open-User-API) backend.

---

### Live Demo: https://openuserprofile.onrender.com

## üöÄ Features

- **Add User:** Add new users with ID, name, username, age, email, and address.
- **View Users:** Display all users in a responsive card layout.
- **Update User:** Edit existing user information inline.
- **Delete User:** Remove users from the system.
- **Notifications:** Success and error messages using `react-toastify`.

---

## ‚ö° Installation

1. **Clone the repository:**

```bash
git clone https://github.com/diwasbk/Open-User-Frontend.git
cd Open-User-Frontend
cd app
```
1. **Install dependencies:**
```bash
npm install
```

---

## Environment Setup

Create a `.env` file in the `app` folder and define the `VITE_BASE_URL` variable with the API URL.  

```env
VITE_BASE_URL=<API_URL_HERE>
```
### *Note:*
*The API URL is available in the [Open User API](https://github.com/diwasbk/Open-User-API) repository
and may be updated from time to time. Make sure to use the latest URL.*
